Built-in methods

This page details the different built-in methods within the fck language.

int type

The int type contains an integer value. Upon initialisation, a value can optionally be assigned. If the value given to be assigned is not an integer, the value is converted into an integer in the following way:

  • float: If the value is a float, it will be rounded to an integer

  • sfloat: If the value is an sfloat, the value of the sfloat will be rounded to an integer

  • bool: If the value is a boolean that is true, this will be converted into a 1. A boolean with a value of false will be converted into a 0

If the value is not covered by the above rules, then an error is raised.
