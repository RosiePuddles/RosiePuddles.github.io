Operators

This page details the different operators available in fck, their use, and return types depending on the given types. The return types will be given in tables, where the leftmost column is the type of the first value or expression given, and the top row is the type of the second value or expression given. This page is grouped into 2 sections; general operators, and operators regarding variable assignment.

Divide, floored divide, division remainder(modulo)

Divides, divides and floors, or returns the remainder of a division of two values. If the second value given for any of these operations if 0, a divide by zero or modulo by zero warning is raised for division and floored division, and modulo operators respectively. A division or floored division by zero results in infinity and the first value or expression being stored along with the result of the infinity. If this value of infinity is later multiplied by zero, the stored expression is returned. A modulo by zero results in 0.

              
                # Divide
                <value_or_expression> / <value_or_expression>
                # Floor divide
                <value_or_expression> // <value_or_expression>
                # Modulo
                <value_or_expression> % <value_or_expression>
