Get Started

To get started with learning fck, see the tutorial section of the website. The tutorials go from "Hello world" to much more complicated tutorials covering almost everything in fck.

Issues and contribution

Since fck is open soure, you are more than welcome to contribute to the development of fck. The GitHub repo is linked at the top of every page, and issues can be found on the issues page of the repo. If you have an issue, before you submit an issue report, consider is it would be better posed as a question or discussion, in which case start a new discussion on the discussion page of the repo.
