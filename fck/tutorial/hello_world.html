Hello World!

fck has two built-in methods for printing items to the screen, print and log. These are fairly similar but are intended for different uses. print is a nicer way of printing items and can be "pretified" with colours (once I impliment it). log on the other hand is intended to be used more for debugging and printing things that don't really need to be "prettified". So, how do we use them? They're really simple to use. Below is a piece of code written in fck that prints "Hello world!" to the screen using the print method and the log method:

            
              print("Hello world!")
              log("Hello world!")
