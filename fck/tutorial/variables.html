Variables

fck has several variable types, int, float, sfloat, bool, string, list, array, and dict. But to start with, we'll only look at int, float, and string types.
To assign a variable, you need to specify the type of the variable, then the name or identifier for that variable, then a set operator followed by a value or expression to assign to the value. To show this, we'll just give you some examples.
To start with, int types store an integer, and only an integer. If you try to set an int type to a non-integer, then it will be rounded to an integer:

            
              >>> int example_variable :: 3
              >>> log(example_variable)
              3
              >>> int example_variable :: -5.2
              >>> log(example_variable)
              -5
              >>>

Next are float types. Floats, short for floating point, store non-integer values like 17.8. they can also store integer values, but they'll be considered to be a float:

            
              >>> float example_variable :: -8.3
              >>> log(example_variable)
              -8.3
              >>> float example_variable :: 12
              >>> log(example_variable)
              12.0
              >>>

Finally, string types store strings. Strings are just pieces of text surrounded by a pair of either inverted commas('), or quotation marks("):

            
              >>> string example_variable :: "Hello world!"
              >>> print(example_variable)
              Hello world!
              >>>

Okay, maybe that wasn't the last bit of this tutorial. Throughout this tutorial, whenever we've been printinting variables, we've always assigned them first, and then printed them, but there is another better way to achieve this, a different assignment operator, :>! This operator is exactly the same as the other assignment operator, but after it's assigned a value to the variable, it returns the value, which means that we can condense our code to the following:

            
              >>> log(float example_variable :> -19.83)
              -19.83
              >>>
