Simple maths

fck has a lot of operators applicable to many different areas. The full documentation for all the operators can be found on the operators documentation page. But for simplicity, we'll just be looking at some simple ones here.
First off, adding or subtracting values is just how you'd expect it, a + and - can add or subtract two values, just like that. Below is a shell that shows how easy adding and subtracting numbers is:

            
              >>> 1 + 2
              3
              >>> 0.05 + 1.03
              1.08
              >>> 12 - 4
              8
              >>> -3 + 3
              0
              >>>

The + and - operators can also be used to indicate whether or not a value is positive or nagetive. Just don't put two positive or negative signs next to each other. That means something entirely different:

            
              >>> +1
              1
              >>> -1
              -1
              >>>

fck also has multiplication and division, big surprise I know, and using them is just as easy as using addition and subtraction:

            
              >>> -5 * 3
              -15
              >>> 12 / 4
              3
              >>> 12 / 0
              *******************************************************
              Warning:
              Divide by zero: Divide by zero found. Returned infinity
              Traceback (most recent call):
                File <shell>, line 1, in <program>
              *******************************************************
              <Infinity>
              >>>

As you can see, you can sort of divide by zero, but also not really. In fck, dividing by zero doesn't stop the code from running, but just returns infinity, which isn't really useful, but at least it doesn't stop everything. This is one of the great features of fck; if it can keep going, it will try it's best to keep going. If you do divide by zero, the result can then be multiplied by zero and you end up back where you started.
